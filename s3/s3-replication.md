# S3 Replication

Replication copies objects between buckets automatically.

## Types

### CRR
Cross-Region Replication.

### SRR
Same-Region Replication.

## Requirements

- Versioning enabled
- Proper IAM permissions

## Benefits

- Disaster recovery
- Backup
- Lower latency

## Important Notes

- Replication is one-way
- Existing objects are not replicated automatically
