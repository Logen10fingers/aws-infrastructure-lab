# Amazon S3 Basics

Amazon S3 is AWS object storage service.

## Core Concepts

### Bucket
Container for objects.

### Object
Actual stored file/data.

## Features

- Highly durable
- Scalable
- Regional service
- Global bucket names

## Common Use Cases

- Backup storage
- Static websites
- Media storage
- Log storage

## Storage Limits

- Object size: 0 B to 5 TB
- Unlimited objects
