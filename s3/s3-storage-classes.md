# S3 Storage Classes

Storage classes optimize cost based on access patterns.

## Types

- S3 Standard
- Standard-IA
- One Zone-IA
- Glacier Instant Retrieval
- Glacier Flexible Retrieval
- Glacier Deep Archive
- Intelligent-Tiering

## Important Idea

Cheaper storage usually means:
- Higher retrieval cost
- Slower access

## Example

- Frequently accessed → Standard
- Rarely accessed → IA
- Archive → Glacier
