# CloudWatch Logs

CloudWatch Logs stores and monitors application and AWS service logs.

## Features

- Centralized logging
- Log filtering
- Metric filters
- Retention settings

## Sources

- EC2
- Lambda
- VPC Flow Logs
- CloudTrail

## Benefits

- Troubleshooting
- Monitoring
- Security auditing
