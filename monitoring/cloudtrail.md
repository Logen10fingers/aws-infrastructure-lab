# AWS CloudTrail

CloudTrail records AWS API activity and account actions.

## What It Tracks

- User actions
- Console activity
- API calls
- Resource changes

## Event Types

### Management Events
Control plane operations.

### Data Events
Object-level actions like S3 uploads.

## Benefits

- Security auditing
- Compliance
- Troubleshooting

## Important Notes

- Enabled by default
- 90-day event history included
